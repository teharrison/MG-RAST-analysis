Using An IPython Notebook show

An IPython notebook is a web based python shell interface with ability to use other shells (bash, perl, R, ruby, etc.). Allows execution of code blocks (called cells) and display of the results as plain text or images or markup (html / javascript).

The notebook page is composed of 2 parts.
  1. A top control bar with drop-down menus
  2. An interactive notebook space below, composed of Ã¢Â€Â˜cellsÃ¢Â€Â™ with input and output areas

The control menus allow manipulation of cells (add, delete, merge, split, move, execute, change type), ouput (expand, collapse, clear), and the kernel that is running the notebook code (interrupt, restart).

Work within the notebook is done by entering commands into a cell, this can be done with one or more lines. To execute the commands in the cell use shift-return. Cells are numbered by execution order and may be re-ran in any order. The results (if any) are displayed in the space below that cell. Result types may be: plain text, JSON or serialized python objects, html, javascript, math equations (latex), images (png, jpg, pdf, etc.)

Using KBase in IPython show

KBase Command Line Scripts

To list available all available KBase command line scripts, run print Ipy.KBASE_CMDS. To view the documentation for a given KBase shell command run ! kb_command_name -h You can also start typing a command in a cell and use the 'tab' key to get a list of available commands or to auto-complete the name. To run a KBase shell command, prepend ! to the front of the command to execute it through the bash shell.


KBase Python Tools

To view the available KBase python classes and functions execute print Ipy.KBASE_IPY. To view the documentation for a given class or function run help(class_or_func_name).
